# Glosario — Task Execution

| Término | Definición operativa |
|---|---|
| WarehouseTask | Raíz que protege las invariantes del módulo. |
| TaskStep | Registro propio del schema del módulo. |
| Tenant | Límite primario de aislamiento y autorización. |
| CorrelationId | Identificador que enlaza API, auditoría, evento y telemetría. |
| Idempotencia | Un mismo identificador produce como máximo un efecto de negocio. |

Los términos ambiguos se resuelven en este archivo antes de modificar FEATURE-TSK-0001.
