# Roadmap

| Fase | Resultado | Gate de salida |
|---|---|---|
| 0 — Fundación documental | Arquitectura, ADR, seguridad, integración, UX, pruebas, operaciones y trazabilidad aprobadas | Gate documental aprobado |
| 1 — Fundación técnica y WMS Core | Plataforma más slices inbound y outbound E2E | Gates H1-H3 superados |
| 2 — Ejecución avanzada | Tareas avanzadas, reposición, olas, devoluciones, calidad, HU/GS1, reconciliación | Piloto estable y casos documentados |
| 3 — Optimización | Slotting, labor, yard, control tower, simulación y shadow mode | Valor operacional medido |
| 4 — 3PL | Multi-owner comercial, contratos, tarifas, portal, SLA y VAS | Segregación y facturación auditadas |
| 5 — Automatización | WES/gateway, AMR, PLC, AS/RS, conveyors e IoT | Necesidad de aislamiento/escala demostrada |

La extracción de un módulo a microservicio requiere evidencia de escalabilidad, aislamiento, equipo, regulación, frecuencia de despliegue o criticidad; no forma parte del MVP.
